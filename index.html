Abstract

  This paper presents a novel adaptive active learning strategy. The most accurate classification model from a heterogenous classifierpool is selected at each learning epoch. Our goal is to acheive greater incremental improvement in accuracy, comparted to using a single type of classification model throughout the process. The startegy is evaluated on benchmark data sets. We further consider an application to texture classification problems un sonar mosaic imagery and compare it with non-adaptive active learning and traditional machine learning approaches.
